# Demo: List Texture Sets

**What we'll build:** A plugin that reads and displays all texture sets in the current project with their resolution.

## What you'll learn
- `substance_painter.textureset` module — reading project state
- Iterating over texture sets with `ts.all_texture_sets()`
- Using `QTextEdit` (read-only) to display output
- **Navigating the documentation** — we'll open Python > Documentation together and find these functions
